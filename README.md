# Political Bias Transfer in LLM-Generated Training Data

This project studies whether political bias from large language models transfers into downstream stance classifiers when those classifiers are trained on synthetic tweets generated by the models.

## Overview

- Compare synthetic data from `GPT-5.4-mini` and `Llama-3-8B-Instruct`
- Train stance classifiers on real data, synthetic data, and mixed data
- Evaluate bias transfer on a held-out real tweet test set
- Measure directional bias, distribution shifts, and calibration differences

## Goals

- Determine whether generator bias changes classifier predictions on real political tweets
- Audit synthetic data for political lean and data leakage
- Analyze whether mixed real+synthetic training reduces bias drift

## Project structure

- `DataGenerator.ipynb`: synthetic tweet generation pipeline
- `data/`: dataset resources and outputs
- `PROJECT_CONTEXT.md`: research plan and methodology
- `.env`: project-local environment variables
- `.gitignore`: repository ignore rules
- `.vscode/settings.json`: workspace-local terminal environment settings
